### 安装
```
 npm install touch-drag-button
```

### 使用
```
import TouchDragButton from 'touch-drag-button';


const app = () => {
  return (
    <TouchDragButton>
      {children}
    </TouchDragButton>
  )
}
```

### 参数
| 参数名 | 类型 | 是否必须 | 说明 | 默认值 |
| :--- | :----  | :--- | :--- | :--- |
| className | string | 否 | 修改悬浮图标初始样式 | 无|
